프로그래밍 개요


1) 범죄율 높은 곳 알림
	1. 범죄 종류별 지역당 건수 검색기능
		KOSIS(국가통계포털)에서 찾은 통계자료들을 통해 어느 지역에서 어떤 범죄가 얼마나  
		발생했는지를 검색할 수 있는 기능이다. 

		출력 예시

		모든 범죄	서울특별시	  수원시		2000년	(	)건	숫자로 보여주거나
		폭행		경기도	  선택 -> 안양시	선택 ->	2001년  (	)건	그래프로 표현
		살인	선택 ->	강원도		  ...			...
		...		...		  평택시		2014년  (	)건
		기타 범죄	전라남도
	
	2. 지역별 범죄율(인구 수와 범죄 건수의 비율) 그래프
		지역별 범죄율을 한눈에 쉽게 보기 위해 그래프를 보여준다. 인구 수가 많은 곳은 범죄 건수가 많으므로
		그에 따른 비율을 계산하여 그래프로 보여주고 범죄율이 가장 높은 곳과 가장 낮은 곳에는 특정
		표시를 해준다. 그래프를 복잡하게 하지 않기 위해 범죄의 종류는 대표적인 범죄들(살인, 절도 등)로
		설정하고 시기는 가장 최근인 것으로 한다.
		
		출력 예시
		
		서울특별시	***************  (  )%
		경기도		*********************  (  )%	제일 높은 곳 표시
		...
		전라남도	*********  (  )%		제일 낮은 곳 표시
	
	3. 위치를 입력받고 위치에 대한 간단한 정보 제공
		사용자의 위치를 입력받아 그 지역에 해당하는 정보를 간단한 수준으로 제공한다. 대부분의
		사람들이 알고있는 또는 겪을 수 있는 범죄들만 분석하고 출력한다.
		
		출력 예시
		
		경기도 수원시  	  입력 -> 경기도 수원시는 2010~2014 강력범죄(폭력) 건수가 높습니다.

2)조사한 각종요소들과 범죄율을 연관
        1.범죄율위험수준과 감소시킬수있는 요소를 알려주는기능
                범죄율을 각 지역마다 확인하여 위험수준을 정하고 위험수준이 높은곳에서 범죄율 감소에 도움을 줄수있는 
                요소들을 출력하고 어느정도 필요한지까지 출력해 주는 기능이다.

 
                출력 예시

                자기 위치를 입력:000

                범죄 수준: 1-5 사이의값이 출력

                범죄감소율에 도움이 될수있는 요소:선택지로 사용 1.가로등
                                                                2.CCTV
                                                                ...
                                              
                000는 XXX개가 더 필요 합니다.->000은 선택지에서 선택된요소

         2.범죄율이 높아질수있는 지역을 알려주는 기능
                범죄율이 아직높지는 않지만 범죄율감소에 영향을 끼치는 요소들이 부족한 지역에 미리 범죄율이 높아질수있다는
                경고를 해주는 기능

                출력 예시
                
                입력된위치:000
 
                000,000 한 요인으로 인하여 이지역은 범죄율이 높아질수있습니다.
